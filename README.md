# Dimensions - ASCII Art Live Identity Reconstruction

**Live Demo:** [https://liviaellen.github.io/Dimensions-ASCII-art/](https://liviaellen.github.io/Dimensions-ASCII-art/)

An interactive art exploration that transforms your live camera feed into dynamic ASCII art with person segmentation, creating a digital reconstruction of your identity through characters, numbers, and animated visuals.

## Features

- **Live Camera Feed**: Real-time webcam capture with person segmentation
- **ASCII Art Generation**: Converts your image into ASCII characters based on brightness
- **Spectrum Color Effects**: Beautiful rainbow/spectrum colors animating across cells
- **Customizable Backgrounds**: Choose between 6 animated GIFs or solid colors
- **Interactive Controls**:
  - Cell size adjustment (15-40px)
  - Color spectrum rotation
  - Wave pattern selection (Radial, Horizontal, Vertical, Sinusoidal, Random)
  - Custom text input for character generation
- **Recording & Export**: Save photos and record videos of your creations
- **Fullscreen Mode**: Immersive viewing experience
- **Stop/Start Camera**: Control camera access with a toggle button

## Technologies

- Vanilla JavaScript
- MediaPipe Selfie Segmentation
- HTML5 Canvas
- CSS3 Animations
